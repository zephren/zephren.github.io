Arrays

Arrays

Arrays are actually quite simple. You quite use them every day without actually realizing it. You can think of literally anything that is a list as an array. You might make a list of strings, a todo list. Or a list of numbers, an expense report. You can easily add an remove items from either of these lists as you either complete items or spend more money. Here are some JS arrays:

var emptyArray = []
var anArray = [1, 2, 3, 4, 5]
var aStringArray = [
    'I',
    'Love',
    'Arrays!'
]
var mixedArray = [
    42,
    'Six',
    777,
    'Why would you do this?',
    3.14
]

Let's use this array for the next for examples:

var expenses = [
    4.78,
    77.89,
    20.50,
    9.99,
    86.90
]

View Items

// This will output the entire array
console.log(expenses)

You can measure arrays:

// .length tells you how many items are in the array
// In the case of expenses, 5
console.log(expenses.length)

You can access items in an array. Items are accessed by their "index". The index is simply a number that indicates an items position in an array, and they always start at 0 for the first item. You can think of the index as the number of items away from the first item it is. So the first item is 0 items away from the first item! The second item is 1 item away from the first item. And so on.

// Using the original array, output each of the items
// There are much better ways of doing this which you'll see soon
console.log(expenses[0])
console.log(expenses[1])
console.log(expenses[2])
console.log(expenses[3])
console.log(expenses[4])

Add/Change Items

You can add items to an array. The term of this is pushing something onto an array. Imagine a stack of things that is just slightly taller than you are, if you want to add something to it, you're going to have to reach up and "push" it on to the array.

// We just made a new purchase, so add it
expenses.push(5.99)
console.log(expenses.length)

You can also change an item at a specific index. Let's add 10.00 to the 9.99 item.

expenses[3] += 10.00

console.log(expenses[3])

Remove Items

Let's say we returned the 77.89 item and we want to remove it from the array. We do this by using the splice function. It's not the best name to describe what it does (though it can do more than remove a single item). Splice removes "y" items, starting from position "x".

array.splice(x, y)

// The item we want to remove is at index 1,
// and we only want to remove the one item
expenses.splice(1, 1)

console.log(expenses.length)
console.log(expenses)

If you specifically want to remove the last item from the array, you can "pop" it off. Imagine that it is stuck on the array and when you remove the item at the last index, it "pops" off.

var poppedItem = array.pop()
console.log(poppedItem)
