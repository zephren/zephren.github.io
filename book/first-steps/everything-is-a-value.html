Everything is a Value

Everything is a Value

The title of this section is a bit of an over generalization, but not by much. What I'm trying to convey is that almost anything in JavaScript can be assigned to a variable. So technically, almost everything is a value. I'll give a few examples here with things that have been covered up to this point. But the best way to figure out what is and is not a value, and what you can do by assigning it to a variable, is to try it.
