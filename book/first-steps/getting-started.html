Getting Started

Before getting into anything else, let's first setup a very simple development environment. There are tons of tools available on the internet for testing out JS; however, many of them incorporate HTML and CSS. We will focus on those later and don't need to worry about them right now. For now I will recommend that you use the following site/tool:

https://jscomplete.com/repl

This page on this site let's you only write and run JS.

Once you've got that opened up type put the following into the left editor area:

console.log('My first line of JavaScript');

Then click he play button or press Cmd/Ctrl + Enter. This will run the JS you have in the left column. And you will be able to see the output in the right column.

Going forward, any place that you type code will be called "your editor" and the place where you see output will be aptly called "your output" or "the console" (yep, that's related to the console.log from above).
